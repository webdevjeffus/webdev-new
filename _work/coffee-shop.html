Instructions

Create an order record by filling in the fields under "Create a new order," then clicking the "Place order" button. You can read all current records in the "Orders in Process" table.

Update any existing record by clicking the "Change Order" button, entering your changes, and then clicking the "Save" button. You may delete any existing record simply by clicking the "Close Order" button.

You are welcome to add, edit, or delete any records in this app. The database is cleared every 24 hours, so it may be empty when you arrive. If that's the case, feel free to populate it!

About this app

This is a complete, working CRUD app, connected to a persistent database through a ReSTful API. It uses jQuery and AJAX to interact with the database, and all changes are displayed on the page without a refresh.

The API used by this app, Learning to ReST, is provided by LearnCodeAcademy, a YouTube channel offering coding tutorial videos.
