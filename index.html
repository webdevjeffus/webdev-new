Welcome to the portfolio and blog of Jeff George, an independent full-stack web developer working out of New York City, NY.

This site is a work in progress. It is being developed using the Jekyll static site generator, along with my own drop-in.css stylesheet and griddy responsive grid system. When complete, it will replace my existing, vanilla HTML/CSS website at webdevjeff.us.

About this website

This current version of my professional website is maintained using the Jekyll static-site generator, which automates the assembly of each page in the site based on templates and content I have created specifically for this site. Jekyll uses the Liquid scripting language to bring programming concepts such as flow control, variables, and collections into HTML. You can explore this site's unprocessed Jekyll source code in a repo on my public Github account.

The general design and basic styling for the site is provided by drop-in.css, a robust stylesheet I created that instantly styles any web app with a single line of HTML code. The particular look for this site comes from drop-in's Whiteboard theme, which I created for this website. The responsive functionality of the site comes from griddy.css, a lean, simple, mobile-first grid system that I developed to avoid the overhead and cookie-cutter look of a framework like Bootstrap or Foundation. You can learn more about these tools at my Github account, and I will blog about how they are used in this website soon.

The original concept for the site began with a mood board, followed by mock-ups prepared using the Balsamiq Mockups 3 wireframing package. Images were prepared using GIMP, a powerful open-source alternative to Adobe Photoshop. Fonts include Permanent Marker (banner headings), Walter Turncoat (sub-headings), and Open Sans (body text), which are all served by Google Fonts. Many icons, including social media buttons, are displayed using Font Awesome.
